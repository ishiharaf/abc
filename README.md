# ABC Tools

An extension to preview and play music written in ABC notation inside Visual Studio Code. It's powered by the [abcjs](https://www.abcjs.net/) library.

## Usage

Open the command palette and type `ABC` and search for the `ABC: Show Preview` command. It'll open a new panel with a preview of the ABC file. Clicking on the `‚èµ` icon will play the file.
